May 17, 2018

I did some more research. I updated my kernel using dnf update kernel command. Installed the kernel-devel package for my distribution kernel. Then compiled the hello kernel module. It worked and then ran insmod ./hello-kernel.ko as root. It gave error message that permission denied. I researched more and found that to load module using insmod the secure boot has to be turned off. Did that in BIOS menu and then the module successfully loaded using insmod. Then, I tried to find the kernel log at /var/log/messages. The 'messages' file was not their and then I read the README in /var/log to find that journalctl keeps all the log files. I checked the journalctl and found the "Hello, World!" message when the module was loaded. Delighted to write my first kernel module. And I know it only gets tougher from here. I am ready.
