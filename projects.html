Automated Internet Speed Visualiser

July - 2018

Tech Stack : Python (selenium, pandas, matplotlib)

This is a project to automatically test internet speed from speedtest.net every 5 minutes, extract data from it by their xpaths and store as Excel file. Then plot a time series graph to visualise the Upload and Download speed trends. The script is resilient to network problems and it can bounce back even in very bad internet speeds.
Github link : Internet Speed Visualiser

Selenium-python browser automation

March - 2018

Tech Stack : Python (selenium, pandas)

This automation project is to automate some repetitive works done by the office of Assistant Commercial Manager(ACM), Khurda Road Division, Indian Railways. This script downloads two files from FOIS website and merges them. It then extracts a field from more than 100 RR's from the website one by one. This whole thing needs more than 1 hour time to do each working day by an employee. But, by using this script the whole work is done within 10 minutes without any human intervention. This saves about 400 hours of time per year for ACM Office.
Github link : Browser automation
